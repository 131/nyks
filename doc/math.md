## log2

Returns the base 2 logarithm of a number.

```javascript
const log2 = require('nyks/math/log2');

log2(2); // return 1
```

------

<a name="log10"></a>
## log10

Returns the base 10 logarithm of a number.

```javascript
const log10 = require('nyks/math/log10');

log10(10);  // return 1
log10(100); // return 2
```
